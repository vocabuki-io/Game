# アイデアメモ — ターン制プリズンブレイク（脱獄）ゲーム

> ステータス: 吐き出し（ブレスト）フェーズ。順不同・断片OK。あとで設計書に整理する。

## コンセプト（軸）
- **対戦ゲーム**：囚人 vs 看守
- ゴッフィ（ゴッドフィールド）のように
  - 簡単に遊べる
  - ルールが分かりやすい
  - 最後まで勝者が分からない（逆転性・緊張感）

## 基本システム
- **行動カード**：ランダムに割り振られる（手札運の要素）
- **イベント**：手荷物検査 / 工事 / 刑務作業 / 面会
- **2Dマップ**：すごろく風のマス移動

## 勝利条件

### 囚人の勝ち
- 複数ある脱獄方法のいずれかを達成して脱獄する
- 武器パーツを集めて看守を倒す

### 看守の勝ち
- 20日（仮）後の期日まで囚人を監督し、期日に処刑する（＝逃さず時間切れ）
- 規定を破った囚人を倒す（※模範囚は倒せない）

## 未確定・要検討（後で詰める）
- 人数：囚人1 vs 看守1？ 複数囚人 vs 看守？ 何人まで？
- 「規定」とは何か（破る/守るの判定条件、模範囚の定義）
- 脱獄方法の種類（トンネル / 変装 / 面会経由 / 賄賂 …）
- 武器パーツの入手経路（イベント/マス/カード）
- ターンの流れ（1日=1ターン？ フェーズ構成）
- 情報の非対称性（看守は囚人の手札や位置をどこまで見えるか）
- 運と戦略のバランス（カード運 vs 位置取り）
